# Sync-On: IDE Configuration Sync

Sync-On helps you keep your VS Code (and compatible forks) configuration synchronized across multiple machines using a private GitHub Gist.

## Features

- **Authenticated Securely**: Uses your GitHub account to store data privately.
- **Settings Sync**: Synchronizes your `settings.json`.
- **Keybindings Sync**: Synchronizes your `keybindings.json`.
- **Extensions Sync**: Keeps your list of installed extensions in sync.
- **Auto-Sync**: Automatically uploads changes when you modify settings or install extensions.

## Usage

1.  **Login**: Run command `Sync On: Login with GitHub`.
2.  **Upload**: Run `Sync On: Upload Settings` to push local config to Gist.
3.  **Download**: Run `Sync On: Download Settings` to pull config from Gist.

## Requirements

- A GitHub account.
